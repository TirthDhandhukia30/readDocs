## Philosophy

> tutorials → consumption → forgotten
> 
> readDocs → comprehension → mastery

We don't rewrite documentation. We sequence it.

<br>

| Layer | Choice |
|-------|--------|

## Run It

```bash
npm i && npm run dev
```

That's it. No env files. No database. No auth setup.

<br>
